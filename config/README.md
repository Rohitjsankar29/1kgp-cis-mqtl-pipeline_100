# config/

Generated configuration — **not** committed (see `.gitignore`). Regenerated by
the pipeline:

- `chr22_manifest_95.tsv` — sample manifest (id, https BAM url, pore, mods …),
  written by `scripts/01_extraction/00_make_chr22_manifest.py`.
- `samples_100.tsv` — sample sheet (id → bedMethyl path), built in stage 02.
- `pheno_samples.txt`, `keep_geno.txt`, `rename_na2gm.txt` — sample lists and the
  GM→NA mapping, built in stage 03.

These are kept out of version control because they contain machine-specific
paths and are deterministically reproducible from the scripts.
